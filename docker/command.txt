#How to pull the new image
docker pull <image_name>

#How to run container
docker run -it <image_name> /bin/bash

#To run the container with detach mode
docker run -d -p <port on which to run:existing port> <image name>

#To see  container
docker container ls

#To see the running container
docker ps

#To delete the all container
docker container prune

#To delete the container once it stopped
docker run --rm -it <image name>

#Copy a jar file into the container
docker container cp target/<jar_name.jar> <container_name>
Example
docker container cp target/hello-world-rest-api.jar naughty_knuth:/tmp

How to make the image first using the spring boot app
1) create the spring boot app
2) create the docker file
3) create the docker image : docker build -t <image name> . // . means where the Docker file located
4) run your docker image  : docker run -p <port on which to run:existing port> <image name>

--------------------------------------------------------------------------------------------------------------------------------
How to push the image to docker hub
1) First we need to assign the tag
docker tag <image name> <repository name : tag name>
example
docker tag spring-boot-app jagdishkamadi00/my-first-repository:spring-boot-app

2) Push the image
docker push jagdishkamadi00/my-first-repository:spring-boot-app
