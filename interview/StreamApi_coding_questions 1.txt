PROBLEM STATEMENT-8:
Reverse a string with speical characters and Speical characters position shouldn't be changed

String input = "a,b$c";
        
        List<Character> charList = input.chars()
                .mapToObj(c -> (char) c)
                .filter(Character::isLetter)
                .collect(Collectors.toList());


        AtomicInteger atomicInteger = new AtomicInteger(charList.size() - 1);

        String result = input.chars()
                .mapToObj(c -> (char) c)
                .map(c -> Character.isLetter(c) ? charList.get(atomicInteger.getAndDecrement()) : c)
                .map(String::valueOf)
                .collect(Collectors.joining(""));

        System.out.println(result);

   OutPut:

   15 -16
   10 -16
   16-20
   20-50
   8-9
   9-50
   7-50
   50 - Max Integer

SOLUTION : 
           int[] intArray = {15, 10, 16, 20, 8, 9, 7, 50};

           for (int i = 0; i < intArray.length; i++) {
            for (int j = i + 1; j < intArray.length; j++) {
                if (intArray[j] > intArray[i]) {
                    System.out.println(intArray[i] + " - " + intArray[j]);
                    break;
                }
            }
        }
        System.out.println(intArray[intArray.length - 1] + " - Max Integer");
